Chrome Or Edge

  1. Download and extract the Chrome/Edge ZIP.
  2. Open chrome://extensions or edge://extensions.
  3. Enable developer mode.
  4. Choose Load unpacked.
  5. Select the extracted extension folder.

Firefox

  1. Download and extract the Firefox ZIP.
  2. Open about:debugging#/runtime/this-firefox.
  3. Choose Load Temporary Add-on.
  4. Select the extracted manifest.json.
  5. Grant site access from the extensions button if needed.

Backend Requirement

The extension defaults to the hosted CMS Smartling backend at https://usifhqtsagrqt01.melaleuca.net/cms-smartling. Use the extension popup if you need to change or test the backend URL.

Updates

To update, download the latest ZIP, extract it, and reload the unpacked extension from the same browser extension page.
