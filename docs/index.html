CMS Smartling Assistant

Browser extension downloads for CMS product-copy translation workflows.

Download The Extension

Choose the ZIP for your browser, extract it locally, and load the unpacked extension. Redownload the ZIP when a new version is published.

Download Chrome/Edge ZIP Download Firefox ZIP

Chrome Or Edge

  1. Download and extract the Chrome/Edge ZIP.
  2. Open chrome://extensions or edge://extensions.
  3. Enable developer mode.
  4. Choose Load unpacked.
  5. Select the extracted extension folder.

Firefox

  1. Download and extract the Firefox ZIP.
  2. Open about:debugging#/runtime/this-firefox.
  3. Choose Load Temporary Add-on.
  4. Select the extracted manifest.json.
  5. Grant site access from the extensions button if needed.

Backend Requirement

The extension talks to the local CMS Smartling backend at http://127.0.0.1:17817 by default. Use the extension popup to change or test the backend URL.

Updates

To update, download the latest ZIP, extract it, and reload the unpacked extension from the same browser extension page.
